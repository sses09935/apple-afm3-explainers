# AFM3 Wiki Layer

## Purpose

This directory contains navigation and maintenance aids for maintainers and AI agents.

It does not introduce new technical facts.

## Authoritative Sources

Authoritative facts remain in:

- `content/knowledge-base.md`
- `sources/source-index.md`
- `sources/source-map.md`

## Files

- `claim-index.md` — claim → source → status → KB → audience → language → diagram navigation.
- `source-authority-map.md` — how source authority layers are interpreted.
- `status-dashboard.md` — update-risk tracking for `official-beta`, `forthcoming`, `reported-excluded`.
- `bilingual-term-map.md` — Traditional Chinese ↔ English terminology alignment.
- `audience-route-map.md` — how one factual base is reframed for three audiences and two languages.

## Rule

If any wiki file conflicts with the KB or source files, the KB and source files win.
